/filter

Filter events by:

  • place
  • year
  • magnitude

Place, year and magnitude parameters may be combined, e.g. find all events in Delfzijl between 2005 and 2010:

/filter?place=Delfzijl&min_year=2005&max_year=2010

Place

Parameters

  • place: string (optional)

Example

/filter?place=Groningen

Year

Parameters

  • min_year: integer (optional)
  • max_year: integer (optional)

Example

/filter?min_year=2005&max_year=2010

Magnitude

Parameters

  • min_mag: float (optional)
  • max_mag: float (optional)

Example

/filter?min_mag=1.0&max_mag=3.0
